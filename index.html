Lecture 1: NPV & IRR

Fundamentals of Financial Valuation

1. Net Present Value (NPV)

EXCEL CODE
=NPV(discount_rate; range_of_CF) (if t=1)
= Initial_CF + NPV(rate; range) (if t=0)
  • NPV measures the value created by an investment.
  • Decision Rule:
    • If NPV > 0 → Accept
    • If NPV < 0 → Reject

2. Internal Rate of Return (IRR)

EXCEL CODE
=IRR(range_including_t0)
Check: NPV(IRR, range) = 0
  • IRR is the discount rate that makes NPV equal to zero.
  • It is the expected annual percentage yield of the investment.

3. Loan Table (Amortization)

EXCEL CODE / CALCULS
  • Beginning Principal (n) = - Initial Investment
  • Payment (at end of year) = Cash Flow
  • Interest each year = Beginning Principal * IRR
  • Principal repaid = Payment - Interest
  • Ending principal = Beginning Principal - Principal Repaid
  • The table shows how each payment (Cash Flow) is split between interest and principal repayment.
  • Using the IRR ensures the loan is 100% repaid at the end: Ending Principal = 0.
