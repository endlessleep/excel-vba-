Lecture 1: NPV & IRR

Fondamentaux de l'évaluation financière

1. Net Present Value (NPV)

EXCEL CODE
=NPV(discount_rate; range_of_CF) // (si t=1)
Initial_CF + NPV(rate; range) // (si t=0)
  • Le NPV mesure la valeur créée par un investissement.
  • Règle de décision :
    • Si NPV > 0 → Accept
    • Si NPV < 0 → Reject

2. Internal Rate of Return (IRR)

EXCEL CODE
=IRR(range_including_t0)
Vérification : NPV(IRR, range) = 0
  • L'IRR est le taux d'actualisation qui rend la NPV égale à zéro.
  • C'est le rendement annuel en pourcentage attendu de l'investissement.

3. Loan Table (Amortization)

EXCEL CODE / CALCULS
  • Beginning Principal (n) = - Initial Investment
  • Payment (at end of year) = Cash Flow
  • Interest each year = Beginning Principal * IRR
  • Principal repaid = Payment - Interest
  • Ending principal = Beginning Principal - Principal Repaid
  • Le tableau montre comment chaque paiement (Cash Flow) est divisé entre les intérêts et le remboursement du capital.
  • Utiliser l'IRR garantit que le prêt est remboursé à 100% à la fin : Ending Principal = 0.
