EX 5: FUTURE VALUE (FV)

Calculate the final value of an investment after n periods with compound interest.

Situation: You deposit a sum once and wait without adding anything else. Only an initial investment (PV) exists, with no extra deposits.

Mathematical Formula:

FV = PV × (1 + r)ⁿ

Excel Formula:

=FV(rate ; nper ; 0 ; -PV)

üîÑ 2Ô∏è‚É£ FUTURE VALUE WITH ANNUAL DEPOSITS (Annuity)

Situation: You save the exact same amount every year (e.g., a Savings Plan).

// Full Excel Structure:
=FV(rate ; nper ; pmt ; [pv] ; [type])
  • pmt: The amount you deposit each period (e.g., -100).
  • pv: Your starting capital. If you start from 0, leave empty or type 0.
  • type: Defines the timing of the payment (Crucial).

üéØ Syntax Focus: Skipping Arguments

=FV(rate ; nper ; -deposit ;; 1)

By using the double semi-colon (;;) or double comma (,, depending on regional settings), you tell Excel to skip the PV argument (treating it as 0) while keeping its positional slot, so you can reach the type argument.

THE DUEL: Type 0 vs Type 1

Value Timing Impact on FV
0 (or omitted) End of period Standard calculation. Deposits don't earn interest for the very first period they are made.
1 Beginning of period Because you deposit at the start of the year, the money works for one extra year and generates more interest. FV will be higher.

Sign Convention!

Remember that deposits are cash outflows. You must always put a minus sign (-) in front of the PMT argument, otherwise your Future Value will render as negative.
