üìä 1. UNDERSTANDING STOCK RETURNS

Concept: CAPM works with returns rather than raw prices. It measures how sensitive a stock‚Äôs returns are to movements in the overall market.

Mathematical Formula:
Returnt = ( Pricet / Pricet-1 ) - 1
Excel Formula:
=CurrentPrice / PreviousPrice - 1
Intuition:

This measures the percentage gain or loss between two periods. In CAPM, we focus entirely on the variability (volatility) of these returns as the very definition of Risk. The more it moves, the riskier it is.

‚öñÔ∏è 2. BETA (β) – THE VOLATILITY SENSOR

Definition: Beta measures how sensitive a specific stock is to broad market movements. It asks: "If the market sneezes, does the stock catch a cold?"

Mathematical Formula:
β = Cov(Rstock , Rmarket) / Var(Rmarket)
Excel Formula:
=COVAR(stock_returns ; market_returns) / VARP(market_returns)
Interpretation Guide:
  • β = 1: Matches the market perfectly.
  • β > 1: Aggressive / Highly Volatile (High Risk).
  • β < 1: Defensive / Stable (Low Risk).
  • β = 0: Risk-free (Zero correlation with the market).

üõ°Ô∏è 3. THE CAPM EQUATION

The Core Formula:
re = rf + β × [ E(rm) - rf ]
The Components:
rf (Risk-free rate)

The absolute minimum return you expect just for waiting. Usually represented by government Treasury Bills.

[ E(rm) - rf ] (Market Risk Premium)

The extra reward required for choosing dangerous stocks over a perfectly safe bank account.

FINAL RESULT

This re is your Cost of Equity. It serves as your Discount Rate to value the share!

4. EXCEL REGRESSION SUMMARY OUTPUT

A step-by-step technical guide to generating and reading the regression table to estimate Beta from raw returns.

  1. Prepare two columns: Market returns (Independent X) and Stock returns (Dependent Y).
  2. Activate Toolpak: If missing, go to File > Options > Add-ins > Manage Excel Add-ins > Go > Check "Analysis ToolPak".
  3. Run Regression: Go to Data > Data Analysis > Regression.
    • Input Y Range: Stock returns (The asset you are measuring)
    • Input X Range: Market returns (The baseline index)
Reading the Summary Output
1. Regression Statistics:

Focus on R-Square. It shows the percentage (%) of the stock's movement that is directly explained by the market's movement.

2. Coefficients Table (CRITICAL):
  • Intercept = Alpha (α)
  • X Variable 1 = Beta (β) This is your primary target.

EXAM TIP

If the regression output Coefficients Table gives you a P-value > 0.05, it means your Beta might not be statistically significant! The market does not reliably explain this stock.

EXTENSION: RISK STRUCTURE & VOLATILITY

üìâ 1. TOTAL RISK (SIGMA σ) & VOLATILITY

What is Sigma (σ)?

Definition: The standard deviation of returns. It measures how much returns fluctuate around their average.

Financial Meaning:
Higher σ → More volatile → More risky
Lower σ → More stable → Less risky

Logic: Sigma measures Total Risk (Market risk + Firm-specific risk).

Monthly Volatility in Excel
=STDEV(range_of_monthly_returns)
CRUCIAL RULE:

Always use Returns, never Prices.

Interpretation: If σmonthly = 0.05, it means monthly returns typically fluctuate by about 5% around their average.

Annualizing Volatility (The √12 Rule)
σannual = √12 × σmonthly
Excel Formula:
=SQRT(12) * MonthlySigmaCell

Intuition: Variance scales with time (e.g., 12 months in a year). Since Sigma is the square root of Variance, to annualize Sigma, we multiply by the square root of 12.

üìä 2. FREQUENCY DISTRIBUTION & RISK STRUCTURE

Objective: Group raw returns into intervals (Bins) to visualize the 'shape' of risk (e.g., Symmetry vs. Skewness) and understand how observations are distributed.

The Excel FREQUENCY Function
=FREQUENCY(data_range ; bins_range)
Argument Description
data_range The observed values (returns)
bins_range The upper boundaries of categories

Bin Logic: Excel counts values strictly based on these upper limits, creating intervals like (bin1 , bin2].

Verification & Relative Frequency
Step Excel Formula Important Note
1. Total Check
(Completeness) 		=SUM(frequency_range) // Must equal total observations
2. Relative Frequency
(Percentage) 		=Frequency_Cell / $Total_Cell$ // Notice the Absolute Reference ($)

üìê COMPARISON RECAP

Sigma (σ)

Measures Total Risk.
Includes both Systematic (Market) risk and Unsystematic (Firm-specific) risk.

Beta (β)

Measures Systematic Risk only.
Captures ONLY the sensitivity to broad market movements.

EXCEL TIP: ARRAY FUNCTIONS

FREQUENCY is an array function. In older Excel versions, simply pressing Enter won't work. You must select the entire output range first, type the formula, and press Ctrl + Shift + Enter to apply it correctly!
